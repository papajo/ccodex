# ccodex v0.6 Sprint Plan — Diff-based edits

Goals:
- Generate and display diffs for proposed file changes
- Ask for confirmation before applying patches

Planned tasks:
1. Implement patch generator that outputs unified diffs for proposed edits
2. Add `/apply-diff` command which applies an approved diff
3. Integrate diff preview into `/write` command flow
4. Add tests for diff generation and patch application

Acceptance criteria:
- Diffs are human-readable and apply cleanly via pure-python patch application
- No destructive writes without explicit confirmation
