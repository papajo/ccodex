# ccodex v0.5 Sprint Plan — Project awareness

Goals:
- Inspect repository structure and select relevant files for context
- Add commands to summarize project layout and pick files for analysis

Planned tasks:
1. `project-scan` feature to build a lightweight index of repo files (respect `.gitignore`)
2. `/project summary` command that returns a short project layout
3. `/select` command to select a set of files by pattern or heuristic for inclusion in prompts
4. Improve `/prompt` to allow `@file` substitutions for selected files
5. Add tests for file scanning and selection heuristics

Acceptance criteria:
- Project summary lists top-level modules and counts of source files per folder
- File selection works with simple globs and heuristics (language, size)
