# ccodex v0.7 Sprint Plan — Explicit tools

Goals:
- Expose explicit safe tools the assistant may use (read, write, search, git status, run tests)
- Make all tool usage visible and confirmable

Planned tasks:
1. Bundle safe tools: `read_file`, `write_file`, `search_files`, `git_status`, `run_tests`
2. Add `/tools` command to display available tools and their required confirmation flows
3. Add policy gating for destructive tools and secret detection in writes/commits
4. Add tests and documentation for tool confirmations

Acceptance criteria:
- Tools are discoverable and always require user confirmation for destructive actions
- Write operations scanned for potential secrets before applying
