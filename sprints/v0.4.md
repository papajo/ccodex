# ccodex v0.4 Sprint Plan — File awareness + Persistence

Goals:
- Implement safe File I/O commands (`/read`, `/write`) with diff preview and confirmation
- Add persistent conversation history (save/load sessions)
- Add minimal `tests/` coverage for history and config
- Add `/export` conversation export command (Markdown/JSON)

Planned tasks:
1. `/read <path>`: safe read that prints file snippet and asks whether to include in model context
2. `/write <path> --apply-after-confirmation`: produce a unified diff and require explicit confirmation before writing
3. Add environment variable `CCODEX_HISTORY_PATH` and implement history save/load
4. Add `tests/test_history.py` and CI-friendly test runner instructions
5. Add `/export [--format json|md] [--path PATH]` to dump current session
6. Update README with new CLI commands and examples

Acceptance criteria:
- `/read` and `/write` commands exist and require explicit confirmation before writing
- Conversation history persists across runs when `CCODEX_HISTORY_PATH` is set
- Tests run locally: `pytest tests/`
